# meile-plan-rest
Meile Subscription Plan REST API

## Dependencies
* uWSGI, `sudo apt install uwsgi-plugin-python3`
* Flask, 
* FlaskMySQL, 
* pexepct, 
* PyJWT, 
* date-util 
* Werkzeug, 
* SQLAlchemy,

`pip install -r requirements.txt`
